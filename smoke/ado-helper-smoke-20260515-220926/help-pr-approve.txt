Approve a pull request (vote = 10, configurable via --vote)

Usage: ado pr approve [OPTIONS] <ID>

Arguments:
  <ID>
          Pull request ID

Options:
      --org <ORG>
          Override the organization URL from config (e.g. https://dev.azure.com/myorg)

      --vote <VOTE>
          Vote value: 10=approve, 5=approve with suggestions, 0=no vote, -5=waiting, -10=reject
          
          [default: 10]

      --project <PROJECT>
          Override the default project from config

      --repo <REPO>
          Repository name (defaults to ADO_REPO env var or origin remote)

      --output <OUTPUT>
          Output format

          Possible values:
          - text:  Plain text, one item per line (default)
          - json:  JSON — full API response, useful for scripting
          - table: Column-aligned table (only meaningful for list commands; falls back to text elsewhere)
          
          [default: text]

  -h, --help
          Print help (see a summary with '-h')
