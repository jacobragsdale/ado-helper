Create a new pull request

Usage: ado pr create [OPTIONS] --title <TITLE>

Options:
      --org <ORG>
          Override the organization URL from config (e.g. https://dev.azure.com/myorg)

      --title <TITLE>
          Pull request title

      --project <PROJECT>
          Override the default project from config

      --source <SOURCE>
          Source branch (defaults to current git branch)

      --output <OUTPUT>
          Output format

          Possible values:
          - text:  Plain text, one item per line (default)
          - json:  JSON — full API response, useful for scripting
          - table: Column-aligned table (only meaningful for list commands; falls back to text elsewhere)
          
          [default: text]

      --target <TARGET>
          Target branch
          
          [default: main]

      --description <DESCRIPTION>
          Description / body text

      --draft
          Create as a draft PR

      --reviewers <REVIEWERS>
          Comma-separated list of reviewer display names or emails

      --repo <REPO>
          Repository name (defaults to ADO_REPO env var or origin remote)

      --field <NAME=VALUE>
          Generic field set, repeatable. e.g. --field isDraft=false

  -h, --help
          Print help (see a summary with '-h')
