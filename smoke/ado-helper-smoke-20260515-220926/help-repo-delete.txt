Delete a repository (permanent — there is no recycle bin)

Usage: ado repo delete [OPTIONS] <NAME>

Arguments:
  <NAME>
          Name (or ID) of the repository to delete

Options:
      --org <ORG>
          Override the organization URL from config (e.g. https://dev.azure.com/myorg)

      --yes
          Required confirmation — this is permanent

      --project <PROJECT>
          Project the repo belongs to (defaults to configured project)

      --output <OUTPUT>
          Output format

          Possible values:
          - text:  Plain text, one item per line (default)
          - json:  JSON — full API response, useful for scripting
          - table: Column-aligned table (only meaningful for list commands; falls back to text elsewhere)
          
          [default: text]

  -h, --help
          Print help (see a summary with '-h')
